Docker

Functions for making a docker image for an experiment run.

Functions:

build_docker(experiment_name, cache_folder, …)

Runs the docker build command.
curifactory.docker.build_docker(experiment_name, cache_folder, version)

Runs the docker build command. This should output the progress to the console as it runs.

Note

This assumes your dockerfile is at docker/dockerfile.

Parameters

  • experiment_name (str) – The name of the experiment that was run.

  • cache_folder (str) – The folder to grab cached objects from.

  • version (str) – A run number/timestamp.
